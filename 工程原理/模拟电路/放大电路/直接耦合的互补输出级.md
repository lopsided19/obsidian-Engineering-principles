## 功放

这三类功放（功率放大器）的区别，本质上就是三极管“静态工作点（Q点）”选在哪里的问题，也就是管子在没有信号时，到底开多大的门。

为了方便理解，我们引入一个核心概念：**导通角（$\theta$）**。它代表在一个完整的周期（$360^\circ$）里，管子有多少时间是在一边流汗一边干活的。

### 1. 甲类功放 (Class A)：全时工作的“高富帅”

甲类功放的工作点选在交流负载线的**中心**。

- **工作状态：** 管子在整个信号周期内（$360^\circ$）一直导通。哪怕没有信号输入，它也流着巨大的静态电流。
    
- **优点：** **线性度完美**。因为它完全避开了死区，没有开关噪声，音质最纯净，被称为发烧友的终极追求。
    
- **缺点：** **效率极低**。由于不干活也耗电，大部分能量都变成了热量。理论最大效率只有 **25%**（接变压器耦合可达 50%）。
    
- **形象比喻：** 像是一台永不熄火的赛车，哪怕停在路边等红灯，引擎也在全速轰鸣。


### 2. 乙类功放 (Class B)：按需分配的“极简主义”

工作点正好选在**截止点**（$V_{BE} = 0$）。

- **工作状态：** 像我们之前讲的互补输出，一个管子管半周（$180^\circ$）。没信号时，管子处于完全关断状态，静态电流为 0。
    
- **优点：** **效率高**。只有有信号时才耗电，理论最大效率可达 **78.5%**。管子发热小，体积可以做得很轻便。
    
- **缺点：** 存在致命的**交越失真**。因为信号在过零点切换时，管子需要时间“苏醒”，导致波形断裂。
    
- **形象比喻：** 像是两个接力运动员，只有棒子传到手里才跑，但因为交接棒动作太慢，每次都会耽误一点时间。

### 3. 甲乙类功放 (Class AB)：中庸之道的“聪明人”

这是目前工程应用（包括大部分音响、驱动器）中最常见的类型。工作点选在**稍稍开启**的位置。

- **工作状态：** 给管子一点微小的偏置电压（前文说的“定金”），让导通角**略大于 $180^\circ$**。
    
- **优点：** 完美解决了乙类的交越失真，同时又保留了接近乙类的高效率。它是性能与功耗最平衡的选择。
    
- **缺点：** 比乙类稍微费一点电（有微小的静态电流），电路设计比乙类稍复杂（需要偏置电路）。
    
- **形象比喻：** 还是那两个接力运动员，但在交班前，接棒的人会提前跑起来一段，实现“无缝对接”。

简单直接的回答：**它是“互补直接耦合放大电路”，但它不是我们通常所说的“互补输出级（功率放大级）”。**


### 1. 为什么它是“互补直接耦合”？

- **直接耦合：** 你看 $T_1$ 的集电极和 $T_2$ 的基极之间没有电容，这就是典型的直接耦合。
    
- **互补：** $T_1$ 是 **NPN** 型三极管，$T_2$ 是 **PNP** 型三极管。这种“一正一负”的搭配就叫互补管。
    

### 2. 为什么它不是“互补输出级”？

在模电教材（比如华成英老师的版本）中，“互补输出级”是一个专有名词，特指**推挽输出（Push-Pull）**。

- **工作方式不同：**
    
    - **推挽（你之前问的那种）：** 两个管子并列，一个管正半周，一个管负半周，像接力赛。
        
    - **这张图（级联）：** $T_1$ 负责把信号放大后，丢给 $T_2$ 接着放大。$T_2$ 是全程参与工作的，它属于**单管放大**（甲类状态），而不是接力赛。
        
- **主要任务不同：**
    
    - **输出级：** 为了带大负载，追求大电流、高功率、高效率。
        
    - **这张图：** 它的核心任务是“电位移动（Level Shifting）”。
        


**一句话总结：** 这张图里的 $T_1$ 和 $T_2$ 虽然是“互补”的，但它们是在**谈恋爱（级联放大）**，而不是在**合伙搬砖（推挽功率输出）**。


## 互补输出定义
“互补输出”级（Complementary Output Stage）通常也被形象地称为**推挽（Push-Pull）输出**。


所谓互补，是指使用了两只**极性相反**的三极管（或场效应管）：

- **NPN 型管（或 N 沟道 MOS）**：像一个向上推的杠杆，负责信号的正半周。
    
- **PNP 型管（或 P 沟道 MOS）**：像一个向下拽的钩子，负责信号的负半周。
    

它们共用一个输入信号，但在物理特性上一个“喜正”，一个“喜负”，正好拼凑成一个完整的正弦波。

## OCL 互补输出级电路图

这是最经典的直接耦合互补输出级（OCL 电路）。它省去了大电容，直接把负载接到地。

```
          +Vcc (正电源)
           │
      ┌────┴────┐
      │      c┌─┴─┐
输入信号 ───── ┤ T1│ (NPN) —— "推" (Push)
  vi  │      e└─┬─┘
      │         │
      │         ├───输出信号 vo ───┐
      │         │                 │
      │      e┌─┴─┐              负载
      └────── ┤ T2│ (PNP) —— "挽" (Pull) RL
             c└─┬─┘              │
                │                地
          -Vee (负电源)
```

---

### 它是如何工作的？（接力赛模式）

我们可以把工作过程拆解成两步：

#### 第一步：正半周 ( $v_i > 0$ )

- **T1 (NPN) 导通**：当输入电压升高，T1 的发射结正偏，电流从 $+V_{CC}$ 涌入，经过 T1 流向负载 $R_L$。
    
- **T2 (PNP) 截止**：此时 T2 的发射结反偏，处于“待机”状态。
    
- **动作**：T1 负责把电平**推高**。
    

#### 第二步：负半周 ( $v_i < 0$ )

- **T1 (NPN) 截止**：输入电压变负，T1 关断。
    
- **T2 (PNP) 导通**：此时 T2 的发射结正偏，电流从“地”出发，流过负载 $R_L$ 进 T2，最后流向 $-V_{EE}$。
    
- **动作**：T2 负责把电平**拉低**。

### 意义
没关系，这部分确实是模电从“信号放大”跨越到“能量转换”的逻辑坎儿。咱们不谈枯燥的定义，换成“水龙头”**和**“推车”的直观例子来讲：

#### 1. 为什么单管放大（甲类）效率低且发热？

想象你要用一个水龙头（三极管）来产生一个**正弦波动**的水流。

- **单管的问题：** 水龙头只能“流水”（导通），不能“吸水”（反向导通）。
    
- **为了输出负半周：** 为了让水流能有“下降”的空间，你必须先把水龙头开到**一半**（这就是静态工作点 $Q$）。
    
    - 当信号想往**正**走，你就把龙头开大；
        
    - 当信号想往**负**走，你就把龙头关小。
        
- **代价：** 哪怕此时**没有信号**，水龙头也一直在出水（半开状态）。这些流出来的水没干任何活，全都白白浪费掉了，变成了热量。
    
    - 这就是为什么它的效率最高只有 **25%**——大部分能量都用来维持那个“半开”的状态了。
        

---

#### 2. 互补输出（推挽）为什么高效率？

现在我们换成两个水龙头：一个接在楼顶水箱（正电源），一个接在地下水池（负电源）。

- **没信号时：** 两个水龙头都关得死死的。**电流为 0，不费电，不发热。**
    
- **有正信号：** 楼顶的水龙头打开，往下推水。
    
- **有负信号：** 楼下的水龙头打开，往下拉水。
    
- **结论：** 只有在干活（有信号）的时候才耗电。这种“按需分配”的机制让它的理论效率达到了 **78.5%**。
    

---

#### 3. 什么是“直流偏移低”？（为什么不需要电容）

这是工程实践中最爽的一点。

- **如果是单管放大：** 刚才说了，为了让信号能上下波动，输出端的电位通常被固定在电源电压的一半（比如 $V_{CC}/2$）。
    
    - 如果你直接把 8 欧姆的扬声器接上去，这 $V_{CC}/2$ 的直流电压会产生巨大的直流电流，直接把扬声器的线圈烧冒烟。
        
    - 所以你必须加一个巨大的**隔直电容**来挡住直流，只让交流信号过去。
        
- **如果是互补输出：** 因为正电源和负电源是对称的。
    
    - 一个管子往上顶，一个管子往下拽。
        
    - 当没有信号时，两个管子力量抵消，输出端就像拔河的绳子中间，稳稳地停在 **0V**。
        
    - **0V 意味着什么？** 意味着你可以把扬声器或者电机**直接**接在输出端和地之间，不需要那个笨重、昂贵且会损耗低频信号的大电容。
