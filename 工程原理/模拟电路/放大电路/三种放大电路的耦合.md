# 两级共射放大电路

## 直接耦合

![[09dd3b0122e62ede10b57fcf29b22dad 1.png|747]]

### 1. 电路基本构成

这个电路由两个三极管 $T_1$ 和 $T_2$ 组成，它们都被接成了共射极（CE）组态（发射极都直接接地或通过电位参考点接地）。

- **第一级（左侧 $T_1$）**：由 $R_{b1}$、$R_{b2}$ 和 $R_{c1}$ 组成偏置和负载电路。信号从 $u_i$ 输入。
    
- **第二级（右侧 $T_2$）**：由 $R_{c2}$ 作为负载。
    
- **中间的虚线**：表示两级之间的分界。
    

### 2. 核心特征：直接耦合（Direct Coupling）

你会发现 $T_1$ 的集电极（输出端）和 $T_2$ 的基极（输入端）之间**没有任何电容或变压器**，而是直接相连。这就是“直接耦合”。

- **优点**：
    
    - **低频特性极好**：因为它没有隔直电容，所以它可以放大频率极低、甚至是直流（DC）信号。
        
    - **易于集成**：在芯片（IC）内部，制造大电容非常困难，所以集成电路内部几乎全部采用这种直接耦合方式。
        
- **缺点（挑战）**：
    
    - **静态工作点（Q点）互相影响**：$T_1$ 的集电极电压直接就是 $T_2$ 的基极电压。**如果第一级的工作点稍微漂移一点，第二级就会被剧烈放大**，导致末端输出严重失真。
        
    - **零点漂移（Zero Drift）**：这是直接耦合电路最头疼的问题。温度的变化会引起晶体管参数变化，这种微小的变化会被逐级放大，最后导致输出端在没有信号输入时也有电压波动。

	- **导通电压倍增 ($U_{BE}$)**：
		普通的 NPN 管导通只需要 $0.7V$。但复合管需要同时让两个 PN 结导通，所以总的导通电压 $U_{BE(on)} \approx 0.7V + 0.7V = 1.4V$。



## NPN+PNP

![[5e74904d7478d10ead8bbf598344d94f.png]]
### 1. 核心变化：直流电位的“拉回”

这是这种组合最核心的动机。

- **在全 NPN 电路中**：第一级的集电极电压 $V_{c1}$ 必须大于 0.7V 才能让第二级导通。随着级数增加，每一级的基极电位都会被前一级“抬高”，导致最后一级的集电极电位非常接近电源电压 $V_{cc}$，这大大限制了信号的摆幅（容易过早进入饱和或截止）。
    
- **在本电路（NPN-PNP）中**：
    
    - $T_1$（NPN）输出的电位是往**高**走的。
        
    - $T_2$（PNP）的发射极接在电源高位，它的导通条件是基极电位比发射极**低**。
        
    - 这样，$T_1$ 的输出电位正好可以顺理成章地驱动 $T_2$ 的基极，而 $T_2$ 的输出端（集电极）电位又被“拉回”到了靠近地电位的地方。
        
    - **结论**：这种交替使用 NPN 和 PNP 的方法，可以很好地**中和各级的直流电位**，让电路更容易在低电源电压下获得较大的输出信号摆幅。


### 2. 依然是两级共射（CE-CE）

尽管 $T_2$ 换成了 PNP，但它的基本组态没有变：

- **输入**：信号进入 $T_2$ 的基极。
    
- **输出**：信号从 $T_2$ 的集电极（下方的 $R_{c2}$ 处）取出。
    
- **公共端**：$T_2$ 的发射极通过 $R_{e2}$（上方电阻）接在 $V_{cc}$ 上，对于交流信号来说，$V_{cc}$ 相当于接地，所以它依然是**共射极电路**。

## 阻容耦合
![[aa638d3f7219c5482d737671aa4561bb 1.jpg]]

### 1. 核心改进：直流隔离（隔直通交）

这是阻容耦合电路的精髓。

- **物理机制**：电容“隔直流，通交流”。
    
- **效果**：通过 $C_2$，第一级 $T_1$ 的集电极直流电位被彻底挡住，不会传给第二级 $T_2$ 的基极。
    
- **最大的好处**：**两级的静态工作点（Q点）互不干扰**。你可以独立设计 $T_1$ 的偏置，再独立设计 $T_2$ 的偏置。如果第一级因为温度升高导致电位漂移，这个漂移（属于极低频或直流信号）会被 $C_2$ 挡住，不会引起第二级的连锁反应。这完美解决了“零点漂移”问题。
    

### 2. 组态分析：CE-CC 组合（黄金搭档）

这个电路非常典型，它把我们之前讨论过的两种组态耦合在了一起：

- **第一级 ($T_1$)：共射电路 (CE)**
    
    - 信号从基极入，集电极出。
        
    - **职责**：负责提供高电压增益，把微弱的 $u_i$ 放大。
        
    - 注意 $R_4$ 和 $C_3$：$R_4$ 稳定直流工作点，$C_3$（旁路电容）在交流状态下把 $R_4$ 短路，确保交流增益不会被削弱。
        
- **第二级 ($T_2$)：共集电路 (CC / 射极跟随器)**
    
    - 信号从基极入，**发射极出**（注意输出 $C_4$ 接在发射极上）。
        
    - **职责**：阻抗匹配。利用共集电路“高输入阻抗、低输出电阻”的特性，作为输出缓冲级，增强带负载 $R_L$ 的能力。


## 三种电路的耦合

### 一、 共射（CE）与共集（CC）的耦合

共射和共集的组合是模拟电路中最常见的搭配，主要有两种玩法：

#### 1. 共射 $\rightarrow$ 共集（CE-CC 组合）：完美的输出驱动

这是最典型的多级放大模式。

- **分工**：前级用**共射**负责把微弱的电压信号拼命放大；后级接一个**共集（射极跟随器）**。
    
- **为什么这么配**：共射电路的输出电阻很大，带载能力弱。后面挺一个输入电阻极高、输出电阻极小的共集电路，不仅可以完美地把前级放大的电压信号“接过来”（不衰减），还能利用共集极低的输出电阻去驱动低阻抗的负载（比如喇叭或后级大功率电路）。
    

#### 2. 共集 $\rightarrow$ 共射（CC-CE 组合）：高输入阻抗放大器

- **分工**：前级用**共集**，后级接**共射**。
    
- **为什么这么配**：如果你的信号源非常微弱且内阻极大（比如某些高阻传感器），直接接共射会把信号源拉垮（分压严重）。先用共集电路在最前面“顶住”，利用它极高的输入电阻减少对信号源的索取，然后再把信号送给共射电路去进行电压放大。
    

---

### 二、 共基电路（CB）也能耦合吗？：高频大杀器

共基电路当然可以耦合！虽然它因为输入阻抗太低，很少单独作为前级，但它和共射、共集组合时，能诞生出电路界非常著名的“明星结构”。

#### 1. 共射 $\rightarrow$ 共基（CE-CB 组合）：大名鼎鼎的 Cascode（共射-共基级联）电路

这绝对是模电教科书里的王牌组合，专门为了解决共射电路的高频恶梦（米勒效应）而生。

- **原理**：信号从共射的基极输入，其集电极直接连到共基的发射极。
    
- **妙在何处**：
    
    - **共射级**：因为共基极的输入阻抗极低，导致共射级的负载电阻接近于 0。根据公式，共射级的电压放大倍数近乎为 $A_{v1} \approx -1$。既然不放大电压，共射级的**米勒效应就直接被彻底抹杀了**！
        
    - **共基级**：共射级虽然不放大电压，但输出了电流。这个电流直接注入共基级的发射极，共基级再利用自己“低入高出”的阻抗特性，在输出端产生巨大的电压放大。同时，共基级本身就没有米勒效应。
        
- **结果**：这个组合**既保留了共射的高电压增益，又拥有了共基极恐怖的宽频带（高频特性）**。在射频（RF）通信、高频放大电路中被极其广泛地使用。
    

#### 2. 共集 $\rightarrow$ 共基（CC-CB 组合）：差分放大电路的灵魂

你可能在学差分放大电路（长尾对电路）时见过它。

- 当差分电路单端输入时，信号从左边三极管的基极进去，从它的发射极出来（这其实就是一个**共集/射随**结构）；
    
- 这个发射极信号直接作为右边三极管发射极的输入，最后从右边管子的集电极输出（这其实就是一个**共基**结构）。
    
- **优点**：这种 CC-CB 的耦合方式同样避开了米勒效应，具有极高的高频响应，且输入阻抗很高。




## 施密特触发器

![[413f91b40c319951f2216e26f9a4e6cf.png]]

标题文字 **“2) 在 $VT_2$ 发射极加稳压管”** 意思是指：为了优化或调整这个施密特触发器的性能，在第二级三极管 $VT_2$ 的发射极（E极）串联了一个**稳压二极管 $VS$**（图中红色部分）。

下面为您拆解这个电路的具体含义和这么做的目的：

- **固定 $VT_2$ 的发射极电位**：当 $VT_1$ 截止、$VT_2$ 导通时，稳压管 $VS$ 工作在反向击穿状态，将 $VT_2$ 的发射极电压牢牢钉在稳压值 $V_{\text{Z}}$。这使得电路的上限翻转电平（阈值电压 $U_{\text{T+}}$）变得非常稳定，不容易受电源电压波动或温度的影响。
    
- **提高电路的开关速度**：稳压管在导通后的动态电阻非常小。相比于传统的公共发射极电阻，它能让三极管在转换状态时提供更大的电流，从而加快电路的翻转速度，使输出波形的前后沿更加陡峭。
    
- **提供偏置通道**：红色的电阻 $R$ 接在电源 $+V_{\text{CC}}$ 和稳压管之间，它的目的是为了给稳压管 $VS$ 提供一个基础的稳定工作电流，确保它随时处于准备好的“稳压状态”。





## 输入电阻和输出电阻
![[98725ffd8c1d9bc8b0452848399537c9 1.png]]

### 输入
好问题，咱们把 $VT_1$ 这一级的输入电阻彻底拆解开，看看为什么公式里有 $R_3$，却“无视”了 $R_2$ 和 $R_4$。

分析输入电阻（$R_i$）时，你要记住一个基本立场：**信号从哪儿进来，我们就从哪儿往里看。**

#### 1. 为什么没有 $R_2$？

输入电阻的定义是：**从输入端（基极 B）看进去，信号电流流向地的路径总阻抗。**

- **信号流向**：信号从基极进来，它有三条路：去 $R_1$、去 $R_5$、钻进三极管。
    
- **$R_2$ 的位置**：它接在三极管的集电极（C）上。对于输入信号电流来说，它得先穿过三极管内部的结，才能到达 $R_2$。
    
- **结论**：在计算**输入电阻**时，我们只关心基极（B）这一侧的“分流”情况。集电极上的电阻 $R_2$ 会影响**电压增益**（决定输出电压多大），但它不直接阻挡信号从基极“挤进去”。所以，输入电阻公式里不带 $R_2$。
    

#### 2. 为什么 $R_4$ 消失了？

这就是图片里那个**电容**（旁路电容）的功劳。

- **交流等效**：在微变等效分析中，这个大电容对交流信号相当于“短路”（一根导线）。
    
- **抢路效应**：发射极电流流出来时，面临两个选择：走 $R_4$ 还是走电容导线？电容导线电阻几乎为 0，电流肯定全走导线。
    
- **结果**：$R_4$ 被彻底“隔离”了，信号根本不经过它。所以在交流通路里，发射极到地之间只剩下了 $R_3$。
    

#### 3. 为什么 $R_3$ 要写进公式，且要乘以 $(1+\beta)$？

这是很多同学最绕的地方。咱们分步看：

##### 第一步：基极看进去的电阻 $R_i'$

输入电流是 $i_b$，基极对地的总电压 $u_i = u_{be} + u_{R3}$。

- $u_{be} = i_b \cdot r_{be}$
    
- $u_{R3} = i_e \cdot R_3 = (1+\beta)i_b \cdot R_3$

##### 第二步：合并计算

把电压加起来：

$$u_i = i_b \cdot r_{be} + (1+\beta)i_b \cdot R_3 = i_b \cdot [r_{be} + (1+\beta)R_3]$$

根据欧姆定律 $R = u/i$，从基极看进去的电阻就是：

$$R_i' = r_{be} + (1+\beta)R_3$$

**这就是 $R_3$ 存在的原因：** 虽然它在发射极，但它阻碍了电流流向地，所以基极信号能“感觉到”它的存在，而且由于 $\beta$ 的放大作用，感觉它**变得巨大无比**。



$$R_{i1} = R_1 // R_5 // [r_{be1} + (1+\beta_1)R_3]$$



### 输出

![[98725ffd8c1d9bc8b0452848399537c9 2.png|872]]

#### 1. 戴维南等效的步骤
**看 $VT_3$ 的组态**：它是一个**共射电路**。

在计算整个电路的输出电阻 $R_o$ 时，根据定义，我们**不包含负载 $R_{12}$ 本身**（它是我们要带的东西，不是电路自带的阻力）。

要算输出电阻，我们需要：

1. **信号源置零**：把输入信号 $\dot{U}_i$ 短路（如果是电流源则开路）。
    
2. **断开负载**：把负载电阻 $R_{12}$ 去掉。
    
3. **加压求流**：在输出端加一个虚拟电压 $U_x$，计算流出的电流 $I_x$，则 $R_o = U_x / I_x$。


- **交流通路**：在交流下，$R_{10}$ 的一端连着集电极，另一端连着 $V_{CC}$（交流地）。
    
- **三极管内部**：从集电极往三极管内部看，内阻 $r_{ce}$ 通常非常大（兆欧级别），在基础分析中通常视作**无穷大（开路）**。

因此，从输出端往回看，你只看到了一条通往交流地的路：**$R_{10}$**。

**结论：**

$$R_o \approx R_{10}$$


#### 2. 为什么老师说直接就是 $R_{10}$？

观察最后一级 $VT_3$：它是一个**共射放大电路**。

从输出端往回看，你会看到两个并联的分支：

1. **分支一**：集电极电阻 $R_{10}$。
    
2. **分支二**：三极管 $VT_3$ 集电极（C极）看进去的内部电阻 $r_{ce3}$。
    
```
       +Vcc (交流接地)
         |
       [ R10 ]
         |
输出端 o--+----[ 三极管 C 极 ]
         |
       ( r_ce3 )
         |
        ...
```

**经验公式推导：**

$$R_o = R_{10} // r_{ce3}$$

在大多数工程计算和模电题目中：

- 三极管的内阻 $r_{ce}$ 通常非常大（几百 $k\Omega$ 到 $M\Omega$ 级别）。
    
- 而集电极电阻 $R_{10}$ 通常只有几 $k\Omega$。
    
- **并联规律**：当一个大电阻和一个小电阻并联时，结果约等于那个小电阻。
    

所以，**$R_o \approx R_{10}$**。这就是老师说的“经验”——在快速估算时，忽略三极管极大的内阻，直接看集电极电阻。

#### 3. 需要注意的“坑”

虽然老师的经验在共射电路里很准，但如果电路变了，结论就会完全不同：

- **如果是射极输出器（共集电极）**：输出电阻会变得非常小，大概是 $[r_{be} + (R_s // R_B)] / (1+\beta)$。

    

分析**输出电阻 ($R_o$)** 时，视角要发生 180 度大转弯。

我们要站在电路的最右端（负载 $R_{12}$ 的位置）往回看。对于整个多级放大电路来说，它的输出电阻就是**最后一级 ($VT_3$) 的输出电阻**。
