### 五大类滤波器的直观理解

按照“允许通过的频带”不同，我们可以把它们想象成不同的**门禁策略**：

#### ① 低通滤波器 (LPF - Low Pass Filter)

- **策略：** 低于截止频率 $f_p$ 的信号可以通过，高于它的全部拦下。
    
- **形象比喻：** 像一个“滤网”，细小的沙子（低频）能漏下去，大的石子（高频）被挡住。
    
- **用途：** 消除高频杂波，提取直流分量。
    

#### ② 高通滤波器 (HPF - High Pass Filter)

- **策略：** 高频信号通过，低频信号（包括直流）被拦下。
    
- **形象比喻：** 像一个“高门槛”，个子够高的（频率够高）才能跨过去。
    
- **用途：** 去除直流偏置（隔直），提取高频脉冲。
    

#### ③ 带通滤波器 (BPF - Band Pass Filter)

- **策略：** 只允许某一个特定频率范围（中间一段）的信号通过。
    
- **形象比喻：** 像一个“窄门”，太胖的（频率太高）和太瘦的（频率太低）都进不来。
    
- **用途：** 无线电收音机选台、特定频率信号检测。
    

#### ④ 带阻滤波器 (BEF - Band Elimination Filter)

- **策略：** 拦住中间特定的一段频率，其余高低频率全部通过。
    
- **形象比喻：** 像路中间的一个“坑”，只有掉进坑里的频率（特定频率）过不去。
    
- **用途：** 消除 50Hz 工频干扰（陷波器）。
    

#### ⑤ 全通滤波器 (APF - All Pass Filter)

- **策略：** 所有频率都能通过，但**相位**会发生改变。
    
- **形象比喻：** 像一条“弯路”，所有人都能走完，但走路的时间（相位）变了。
    
- **用途：** 信号相移、校正。
    

### 3. 有源 vs 无源

你在基础电路里学的 RC 滤波是“无源”的。

- **无源缺点：** 输出电压会随负载变化而大幅波动，且信号越滤越弱（衰减）。
    
- **有源优点：** 1. **有增益：** 信号不仅能滤，还能顺便放大。
    
    2. **高阻输入/低阻输出：** 运放的特性保证了滤波器不会影响前级，也能带得动后级负载。

## 高通滤波器

而对于高通滤波器，就是把电容和电阻换一个位置

#### 低通滤波器 (LPF)

低通的特点是：电容在反馈或并联支路“吸收”高频。

```
           (反馈支路：可以是 Rf，也可以加 C 进行滤波)
                 ________[ Rf ]________
                |                      |
      [ R ]     |      | \             |
Ui ---[###]-----*-----(⁻)  \           |
                       | A  \----------*---- Uo
                *-----(+)  /
                |      | /
              [ C ]
                |
              (GND)
```

_(注：这是同相输入式低通的一种，电容将高频信号短路到地)_

#### 高通滤波器 (HPF)

**位置互换：** 电容 $C$ 移到输入端，电阻 $R$ 移到接地端。

```
           (反馈支路：决定放大倍数)
                 ________[ Rf ]________
                |                      |
      [ C ]     |      | \             |
Ui ---[ | ]-----*-----(⁻)  \           |
      [ | ]            | A  \----------*---- Uo
                *-----(+)  /
                |      | /
              [ R ]
                |
              (GND)
```




## 带通滤波器和带阻滤波器
带通：串联高通和低通
带阻：并联高通和低通
