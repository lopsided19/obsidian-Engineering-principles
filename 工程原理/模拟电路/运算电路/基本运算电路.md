## 反相比例运算电路 (Inverting Amplifier)
![[Pasted image 20260526160156.png|862]]

基于上述法则，我们可以利用基尔霍夫电流定律（KCL）在节点 $N$ 进行分析：


因为 $i_N = 0$，所以流过电阻 $R$ 的电流全部流向反馈电阻 $R_f$：

$$i_R = i_F$$


根据欧姆定律：

$$i_R = \frac{u_I - u_N}{R} = \frac{u_I - 0}{R}$$
    
 $$i_F = \frac{u_N - u_O}{R_f} = \frac{0 - u_O}{R_f}$$
    

联立以上等式：

$$\frac{u_I}{R} = -\frac{u_O}{R_f}$$

最终得出输出电压公式：

$$u_O = -\frac{R_f}{R} \cdot u_I$$

- **比例系数（增益）**: $A_v = -\frac{R_f}{R}$。
    
- **符号含义**: 负号表示输出信号与输入信号相位相反（即**反相**）。
    

### 输入电阻

从输入信号源 $u_I$ 的角度看过去，电路的输入电阻 $R_{if}$ 定义为输入电压与输入电流的比值：

$$R_{if} = \frac{u_I}{i_R}$$

根据前面提到的虚地（Virtual Ground）概念：

- 反相输入端（节点 $N$）的电位 $u_N \approx 0V$。
    
- 因此，输入电流 $i_R$ 实际上全部消耗在电阻 $R$ 上，即 $i_R = \frac{u_I - u_N}{R} = \frac{u_I - 0}{R}$。
    

将 $i_R$ 代入定义式，可以得出：

$$R_{if} = R$$

### 特点

|**特点**|**详细描述**|
|---|---|
|**相位反转**|输出电压与输入电压相位差为 180°。|
|**虚地特性**|反相端 $N$ 的电位始终接近 $0V$，这有利于降低对运放共模抑制比的要求。|
|**输入电阻小**|该电路的输入电阻 $R_i = R$。由于 $R$ 不能无限大（否则噪声会增加），这在需要高输入阻抗的场合是个缺点。|
|**增益可调**|只要改变 $R_f$ 或 $R$ 的阻值，就能非常精确地调整放大倍数。|



## T型网络反相比例运算电路

![[694d5cd0e952c1b92016c00cbbf88243.png|863]]

为了解决一个痛点：**在高增益需求下，避免使用阻值过大的电阻。**

**T型网络的优势**：它利用分流原理，可以用几个较小的电阻（如几十 $k\Omega$）组合出极高的等效反馈电阻，实现上百倍甚至上千倍的放大。

### 电路推导 (利用 KCL 和虚地)

分析这个电路的关键依然是**虚地**（$u_N \approx 0V$）和**虚断**（$i_{in} \approx 0$）。

1. **节点 N 的电流平衡**：
    
    由 $i_1 = i_2$ 可知：
    
    $$\frac{u_I}{R_1} = \frac{0 - u_M}{R_2} \implies u_M = -\frac{R_2}{R_1}u_I$$
    
2. **节点 M 的电流平衡**：
    
    流向 $M$ 点的电流等于离开 $M$ 点的电流之和，$i_2 + i_3 = i_4$（注意图中的电流方向标号）：
    
    $$\frac{0 - u_M}{R_2} + \frac{0 - u_M}{R_3} = \frac{u_M - u_O}{R_4}$$

    整理得：

$$u_O = u_M(1 + \frac{R_4}{R_2} + \frac{R_4}{R_3})$$
    
3. **最终增益公式**：
    
    将 $u_M$ 代入上式：
    
    $$u_O = -\frac{R_2}{R_1} \left( 1 + \frac{R_4}{R_2} + \frac{R_4}{R_3} \right) u_I$$
 
 或者写成更常用的形式：
    $$u_O = -\frac{R_2 + R_4 + \frac{R_2 R_4}{R_3}}{R_1} u_I$$
    

### 总结

- **等效反馈电阻**：
    
    你可以把 T 型网络看作一个巨大的等效电阻 $R_{f(eq)} = R_2 + R_4 + \frac{R_2 R_4}{R_3}$。通过调节 $R_3$（减小 $R_3$），可以显著提升增益，而不需要去动其他电阻。
    
- **输入电阻**：
    
    输入电阻依然是 **$R_i = R_1$**。这和基础反相电路是一致的。
    
- **平衡电阻 $R_5$**：
    
    为了消除偏置电流影响，$R_5$ 的取值应等于同相端看出去的等效电阻：$R_5 = R_1 // (R_2 + R_3 // R_4)$。


## 同相放大电路

![[58ec0091b946dcf636bd0a22cfdbaff1.png|915]]

### 电路推导 (利用虚短与虚断)

我们依然利用理想运放的特性来推导：

- **虚短**: $u_P = u_N$。由于同相端输入为 $u_I$（忽略 $R'$ 上的极小压降），所以 $u_N = u_I$。
    
- **虚断**: 流入反相端 $N$ 的电流为 $0$，因此 $i_F = i_R$。


1. 计算反相端 $N$ 点的电位（其实就是一个分压电路）：
    
    $$u_N = \frac{R}{R + R_f} \cdot u_O$$
    
2. 由 $u_N = u_I$ 得：
    
    $$u_I = \frac{R}{R + R_f} \cdot u_O$$
    
3. 整理得出输出电压公式：

 $$u_O = (1 + \frac{R_f}{R}) \cdot u_I$$
### 输入电阻
同相比例运算电路的**输入电阻**计算方式与反相电路截然不同，这正是它的“成名绝技”。


根据输入电阻的定义：

$$R_{if} = \frac{u_I}{i_P}$$

因为分母 $i_P$ 趋近于 $0$，所以同相比例电路的输入电阻 **$R_{if} \to \infty$**（无穷大）。


### 知识点
#### ① 同相位

输出 $u_O$ 与输入 $u_I$ 的比例系数为 $(1 + R_f/R)$，结果永远为正。这意味着**输出与输入相位相同**。

#### ② 极高的输入电阻

这是同相放大器最大的优势！

- 由于信号直接进入运放的高阻抗输入端，该电路的输入电阻 $R_{if}$ 理论上趋于**无穷大**。
    
- 这使得它对前级信号源几乎没有负载影响，非常适合用于放大内阻很大的传感器信号。

#### ③ 增益限制

- **最小值**: 当 $R_f = 0$ 或 $R = \infty$ 时，$u_O = u_I$，此时电路演变为**电压跟随器**。
    
- 由于公式中存在 “$1 + ...$”，同相比例电路的增益永远 **$\ge 1$**，它无法像反相电路那样实现小于 1 的衰减。

## 共模输入
在分析运算放大器时，“共模输入”是一个非常深入且具有工程意义的考点。简单来说，共模电压就是**运放两个输入端（$+$ 和 $-$）共同承受的电压平均值**。

### 1. 反相比例电路：近乎完美的“虚地”

在**反相比例电路**中：

- **同相端 ($+$)** 直接通过平衡电阻接地，电位基本为 $0V$。
    
- **反相端 ($-$)** 由于“虚短”效应，电位也被锁定在 $0V$ 附近（即虚地）。
    

**结论**：反相比例电路的共模输入电压 **$u_{ic} \approx 0V$**。

- **优势**：运放几乎不需要处理共模信号，因此对运放的共模抑制比（CMRR）要求极低。即使运放本身的性能一般，也不会因为共模电压导致严重的非线性失真。
    

### 2. 同相比例电路：随波逐流的电位

在**同相比例电路**中：

- **同相端 ($+$)** 直接接输入信号 $u_I$，所以其电位就是 $u_I$。
    
- **反相端 ($-$)** 由于“虚短”效应，电位也会紧紧跟随同相端，同样为 $u_I$。
    

**结论**：同相比例电路的共模输入电压 **$u_{ic} \approx u_I$**。

- **挑战**：随着输入信号的波动，运放内部的两个输入管电位都在剧烈跳动。


## 平衡电阻

这里的 $R'$ 就是指的这个东西
![[58ec0091b946dcf636bd0a22cfdbaff1.png|915]]
在运算放大器电路中，**平衡电阻**（也被称为**补偿电阻**）通常接在运放的同相输入端。虽然在理想模型的计算中它似乎“毫无作用”，但在实际电路设计中，它是消除误差的功臣。

### 1. 为什么要用它？（消除“偏置电流”的恶作剧）

实际的运放并不是完美的。它的内部由晶体管组成，为了让这些晶体管工作，必须有微弱的电流流进或流出输入端，这被称为**输入偏置电流 ($I_B$)**。

- **没有平衡电阻时**：
    
    在反相比例电路中，反相端接有电阻 $R$ 和 $R_f$，而同相端直接接地。由于 $I_B$ 流经 $R$ 和 $R_f$ 的并联网络，会在反相端产生一个额外的压降。
    
- **后果**：
    
    运放会把这个由于偏置电流产生的压降当成“信号”进行放大，导致输出端出现一个本不该有的直流电压（**失调电压**）。即便输入为 0，输出也不为 0。
    

### 2. 平衡电阻的原理：以毒攻毒

平衡电阻的作用就是：**在同相端也制造一个完全相同的压降，抵消掉反相端的误差。**

- **操作方法**：我们在同相端串联一个电阻 $R_p$。
    
- **计算逻辑**：
    
    为了让两端的压降相等，我们需要让两端看进去的**等效电阻**相等。
    
    - 反相端看进去的等效电阻是 $R // R_f$。
        
    - 因此，同相端的平衡电阻应取：
        
    
     $$R_p = R // R_f = \frac{R \cdot R_f}{R + R_f}$$

### 总结

**平衡电阻不改变放大倍数，它的唯一使命是：让运放左右两个输入端在“安静（静态）”时感受到的电阻环境一模一样，从而把偏置电流引起的误差减到最小。**

在你看到的教材图中，**反相电路的 $R'$ 和同相电路的 $R'$ 其实都是这个作用**。你现在的实验电路用的是哪款运放芯片？我可以帮你判断一下 $R'$ 对你的精度影响有多大。

## Rp=Rn
![[3ffd6e0da9152ea46762ffc2627c600d.png|758]]

### Rn的误区
这是一个非常容易产生误解的地方。你可能直觉上觉得 $R_1, R_2, R_3$ 是“输入端”，而 $R_F$ 是“反馈端”，它们应该处于不同的地位。

但当我们求平衡电阻（$R'$）时，我们的视角必须切换到运放输入端节点（节点 N）。


在计算平衡电阻时，我们遵循的是**直流等效法则**：

- **信号源 $U_{i1}, U_{i2}, U_{i3}$：** 从直流偏置电流 $I_B$ 的角度看，这些信号源的内阻通常视为 0，即等效接地。
    
- **输出端 $U_o$：** 运放的输出端对地阻抗极低，在直流等效模型中也视为接地。
    

现在，请你闭上眼睛，站在反相输入端（节点 N）这个位置往外看，你会发现：

- 通过 $R_1$ 有一条路通往地。
    
- 通过 $R_2$ 有一条路通往地。
    
- 通过 $R_3$ 有一条路通往地。
    
- 通过 $R_F$ **也有**一条路通往地。
    

这四条路是**各走各的，互不干扰，但终点都是地**。在电路拓扑上，这就叫**并联**。


```
               (运放反相端节点 N)
           ___________|________________
          |           |           |           |
        支路1        支路2        支路3       支路4
          |           |           |           |
        [ R1 ]      [ R2 ]      [ R3 ]      [ RF ]
          |           |           |           |
        (接地)       (接地)       (接地)       (接地)
```

## 仪表放大器

![[8c9487856de1af3c43e90a853c02f0b9.png]]

### 第一张图：电路结构分析

这张图解释了为什么要用三块运放，以及它们各自的分工。

1. **第一级（$A_1, A_2$）：双运放差分输入级**
    
    - **高阻抗：** 信号 $U_{i1}$ 和 $U_{i2}$ 都是直接接到运放的同相输入端。正如我们之前讨论过的，这种方式的输入电阻极高（$\approx \infty$），不会对信号源产生任何负载效应。
        
    - **差分放大：** 这两个运放通过 $R_1$ 和两个 $R_2$ 连在一起，初步放大两个输入信号的差值（$U_{i1} - U_{i2}$）。
        
2. **第二级（$A_3$）：基本减法电路级**
    
    - 它负责把第一级输出的差分信号（$U_{o1}, U_{o2}$）转换成单端输出信号 $U_o$。
        
    - 它还能进一步抑制共模噪声（比如两根导线上同时存在的工频干扰）。
        

### 第二张图：定量推导（利用虚短虚断）
![[fc140888d683f09faecc8b68537c4c65.png]]
#### 1. 第一级的推导逻辑

- **关键点 $U_{R1}$：** 根据“虚短”，$A_1$ 的两个输入端电压相等，所以 $A_1$ 的反相端电压 $= U_{i1}$。
    
    同理，$A_2$ 的反相端电压 $= U_{i2}$。
    
    你会发现 $R_1$ 两端的电压正好就是输入信号之差：**$U_{R1} = U_{i1} - U_{i2}$**。
    
- **电流恒定：**
    
    根据“虚断”，运放输入端不吸收电流。所以从 $U_{o1}$ 经过两个 $R_2$ 和一个 $R_1$ 到 $U_{o2}$ 的这一条路径上，**电流是同一条**。
    
    电流 $I = \frac{U_{R1}}{R_1}$。
    
- **求 $U_{o1} - U_{o2}$：**
    
    这条路径上的总电阻是 $R_2 + R_1 + R_2 = 2R_2 + R_1$。
    
    所以：$U_{o1} - U_{o2} = I \cdot (2R_2 + R_1) = \frac{2R_2 + R_1}{R_1}(U_{i1} - U_{i2})$。
    

#### 2. 第二级的推导逻辑

- $A_3$ 及其周围电阻构成了一个标准的减法器（差动比例运算电路）。
    
- 当满足 $R_3$ 和 $R_4$ 参数对称时（即满足我们说的平衡电阻要求），其输出电压为：
    
    **$U_o = \frac{R_4}{R_3}(U_{o2} - U_{o1})$**。
    
    _(注意图中公式带了一个负号，是因为它取的是 $U_{o1} - U_{o2}$ 的差值)_。
    

#### 3. 最终总增益

把两级公式合并，就得到了仪表放大器的总公式：

$$U_o = -\frac{R_4}{R_3} \cdot (1 + \frac{2R_2}{R_1}) \cdot (U_{i1} - U_{i2})$$


## 电压跟随器

![[348c09d4defb679d771d26412761919a.png]]

### 1. 核心工作原理

电压跟随器本质上是一个**电压串联负反馈**电路。它的核心特点是：**输出电压完全追随输入电压的变化**，即 $u_{\text{O}} = u_{\text{I}}$，电压放大倍数 $A_u \approx 1$。

利用运放的**虚短**和**虚断**概念可以轻松推导：

- **虚短（Virtual Short）：** 运放的同相输入端（$+$）和反相输入端（$-$）电位相等，即 $u_+ = u_-$。
    
- **虚断（Virtual Break）：** 运放的输入阻抗无穷大，流入 $+$ 端和 $-$ 端的电流几乎为 0。


### 2. 两个子图的对比与推导

#### 图 (b) 直连型电压跟随器（最纯粹的形式）

这是最标准的电压跟随器：

- **结构：** 输出端 $u_{\text{O}}$ 通过导线直接 100% 反馈回反相输入端（$-$），因此 $u_- = u_{\text{O}}$。
    
- **推导：** 输入信号直接加在同相端（$+$），所以 $u_+ = u_{\text{I}}$。根据虚短定律（$u_+ = u_-$），直接得出：
    
    $$u_{\text{O}} = u_{\text{I}}$$
    

#### 图 (a) 带电阻的电压跟随器

这个电路在输入端和反馈回路中各串联了一个阻值相同的电阻 $R$。

- **推导：**
    
    - 因为**虚断**，没有电流流入运放的同相端（$+$），所以输入电阻 $R$ 上没有电压降，同相端电位依然是 $u_+ = u_{\text{I}}$。
        
    - 同样因为**虚断**，没有电流流入反相端（$-$），反馈回路的电阻 $R$ 上也没有电流和压降，所以反相端电位依然直接等于输出端，即 $u_- = u_{\text{O}}$。
        
    - 根据虚短（$u_+ = u_-$），最终结果依然是：
        
        $$u_{\text{O}} = u_{\text{I}}$$
        

### 既然计算结果一样，图 (a) 的电阻有什么用？

图 (a) 中加入电阻 $R$ 主要是为了**消除运放输入端偏置电流（Bias Current）带来的静态误差**，也就是实现**阻抗匹配**：

- 从同相端看出去，对地的等效电阻是 $R$。
    
- 从反相端看出去，对地的等效电阻（通过输出端低内阻接地）也是 $R$。
    
- 这样可以使运放两个输入端的静态偏置电流在电阻上产生的压降相等，从而在差分输入端相互抵消，**减小输出端的直流漂移（零点漂移）**。在精密放大或使用双极型（BJT）运放时，图 (a) 的设计更为严谨。



## 反相求和运算电路
![[e18e4c1af0874db562d2217fc2b81343.png]]


利用运算放大器的“虚短”和“虚断”特性，我们可以非常容易地得出它的输出公式。

- **虚地（Virtual Ground）：** 同相输入端（$+$）通过电阻 $R_4$ 接地。由于“虚断”，没有电流流过 $R_4$，所以 $u_+ \approx 0$。根据“虚短”原理，$u_- = u_+ = 0$。因此，反相端节点 **N** 被称为“虚地点”。
    
- **节点电流定律（KCL）：** 根据“虚断”，运放的反相输入端不吸收电流，所以流入节点 N 的所有电流之和等于流出节点 N 经过 $R_{\text{f}}$ 的电流。

1. 各支路电流分别为：
    
    $i_1 = \frac{u_{\text{I}1} - 0}{R_1}$, $i_2 = \frac{u_{\text{I}2} - 0}{R_2}$, $i_3 = \frac{u_{\text{I}3} - 0}{R_3}$
    
2. 反馈支路电流为：
    
    $i_{\text{F}} = \frac{0 - u_{\text{O}}}{R_{\text{f}}}$
    
3. 根据 KCL ($i_1 + i_2 + i_3 = i_{\text{F}}$)：
    
    $$\frac{u_{\text{I}1}}{R_1} + \frac{u_{\text{I}2}}{R_2} + \frac{u_{\text{I}3}}{R_3} = -\frac{u_{\text{O}}}{R_{\text{f}}}$$
    
4. 最终得到输出电压公式：
    
    $$u_{\text{O}} = -\left( \frac{R_{\text{f}}}{R_1}u_{\text{I}1} + \frac{R_{\text{f}}}{R_2}u_{\text{I}2} + \frac{R_{\text{f}}}{R_3}u_{\text{I}3} \right)$$
    

### 电路组件的作用

- **$R_{\text{f}}$ (反馈电阻)：** 决定了增益的总倍数。
    
- **$R_1, R_2, R_3$ (输入电阻)：** 分别决定了每个通道的“权重”。如果 $R_1 = R_2 = R_3 = R_{\text{f}}$，那么 $u_{\text{O}} = -(u_{\text{I}1} + u_{\text{I}2} + u_{\text{I}3})$，这就是标准的**加法器**。
    
- **$R_4$ (平衡电阻)：** 图片下方标注了 $R_4 = R_1 // R_2 // R_3 // R_{\text{f}}$。
    
    它的作用与之前看过的电压跟随器类似，是为了保证运放两个输入端看出去的**等效电阻匹配**，从而抵消偏置电流产生的误差，提高运算精度。

**总结：** 这是一个带加权的、反相的电压求和器。它通过控制电阻比例，精准地实现了 $y = -(a x_1 + b x_2 + c x_3)$ 这样的线性数学关系。




## 同向求和运算电路
![[d0155ab398996eb8bf878bc83802c4a4.png|902]]

这张图片展示的是**同相求和运算电路（Non-inverting Summing Amplifier）**。

与之前的反相求和电路相比，它的输出电压与输入电压的相位相同（同为正或同为负），但在推导和阻抗特性上有所不同。


### 第一步：计算同相端电压 $u_{\text{P}}$

利用**节点电压法**（或叠加原理）。由于“虚断”，没有电流流入运放的 $+$ 端，所以节点 P 的电流满足 $i_1 + i_2 + i_3 = i_4$。

根据公式推导，节点 P 的电压为：

$$u_{\text{P}} = \frac{\frac{u_{\text{I}1}}{R_1} + \frac{u_{\text{I}2}}{R_2} + \frac{u_{\text{I}3}}{R_3}}{\frac{1}{R_1} + \frac{1}{R_2} + \frac{1}{R_3} + \frac{1}{R_4}}$$

将电流展开为电压形式：

$$\frac{u_{\text{I}1} - u_{\text{P}}}{R_1} + \frac{u_{\text{I}2} - u_{\text{P}}}{R_2} + \frac{u_{\text{I}3} - u_{\text{P}}}{R_3} = \frac{u_{\text{P}} - 0}{R_4}$$

整理方程，把含有 $u_{\text{P}}$ 的项移到等号右边：

$$\frac{u_{\text{I}1}}{R_1} + \frac{u_{\text{I}2}}{R_2} + \frac{u_{\text{I}3}}{R_3} = u_{\text{P}} \left( \frac{1}{R_1} + \frac{1}{R_2} + \frac{1}{R_3} + \frac{1}{R_4} \right)$$

最终得出 **$u_{\text{P}}$ 的通用计算公式**：

$$u_{\text{P}} = \frac{\sum \frac{u_{\text{I}i}}{R_i}}{\sum \frac{1}{R_i}} = \frac{\frac{u_{\text{I}1}}{R_1} + \frac{u_{\text{I}2}}{R_2} + \frac{u_{\text{I}3}}{R_3}}{\frac{1}{R_1} + \frac{1}{R_2} + \frac{1}{R_3} + \frac{1}{R_4}}$$

### 第二步：利用同相放大比例计算 $u_{\text{O}}$

此时电路可以看作是一个基础的同相放大器。根据“虚短”和“虚断”，反相端的电压 $u_- = u_{\text{P}}$。

对于由 $R$ 和 $R_{\text{f}}$ 组成的反馈网络，输出电压公式为：

$$u_{\text{O}} = \left( 1 + \frac{R_{\text{f}}}{R} \right) u_{\text{P}}$$

最终结果为：
$$u_{\text{O}} = \left( 1 + \frac{R_{\text{f}}}{R} \right) \cdot \left[ \frac{\frac{u_{\text{I}1}}{R_1} + \frac{u_{\text{I}2}}{R_2} + \frac{u_{\text{I}3}}{R_3}}{\frac{1}{R_1} + \frac{1}{R_2} + \frac{1}{R_3} + \frac{1}{R_4}} \right]$$



## 积分运算电路
![[1318464e26fb648151f4f762c405117d.png]]



###  例题
![[8f2acfba6e98b364397e439d3465a71d.png]]

#### 1. 电路本质：积分器

观察原理图，反馈支路是一个电容 $C$，输入支路是一个电阻 $R$。这是一个**反相积分电路**。

- **输入信号 $u_I$**：是一个 $-3V$ 的直流阶跃信号（由 $0V$ 跳变到 $-3V$）。
    
- **积分作用**：运放会将输入电流积累在电容 $C$ 上。由于 $u_I$ 是负的，根据反相积分公式，输出 $u_o$ 会向正方向线性增加。
    

#### 2. 公式推导过程

图中的推导分为三步，核心逻辑是：**输出电压 = 电容上的积累电荷量 / 电容值**。

1. **积分公式**：
    
    $$u_o = -\frac{1}{RC} \int_0^T u_I dt + u_o(0)$$
    
    因为 $u_I = -3V$ 是常数，且初始电荷 $u_o(0) = 0$，积分后变为：
    
    $$u_o = -\frac{u_I}{RC} T = -\frac{-3}{RC} T = \frac{3}{RC} T$$
    
2. **触发条件**：
    
    题目说电子开关在输出达到 **$+6V$** 时动作。我们将 $u_o = +6V$ 代入公式：
    
    $$6 = \frac{3}{10^4 \times 0.05 \times 10^{-6}} \times T$$
    
3. **计算结果**：
    
    计算分母 $RC = 10^4 \times 0.05 \times 10^{-6} = 0.5 \times 10^{-3}$ 秒（即 $0.5ms$）。
    
    代入公式：$6 = \frac{3}{0.5ms} \times T \Rightarrow 6 = 6000 \times T$。
    
    最后得出 $T = 0.001s = 1ms$。
