## 认识 4位全加器 (74LS283)

全加器的核心逻辑非常简单，就是实现两个 4 位二进制数相加，并考虑低位进位。

- **输入端：**
    
    - 加数 $A$：$A_3 A_2 A_1 A_0$
        
    - 加数 $B$：$B_3 B_2 B_1 B_0$
        
    - 低位进位输入：$CI$（或者 $C_{-1}$）
        
- **输出端：**
    
    - 和 $S$：$S_3 S_2 S_1 S_0$
        
    - 高位进位输出：$CO$


它的数学本质就是执行：

$$S = A + B + CI$$


### 4位全加器应用电路（找规律）

这张图展示的是将**输入信号的某些位同时接到加法器的不同引脚上**。

我们来看看它输入了什么，从而推导它在算什么：

- **$A$ 端输入：** $A_3=0$, $A_2=A$, $A_1=B$, $A_0=C$。所以二进制数 $A = (0ABC)_2$。
    
- **$B$ 端输入：** 连线错位了。$B_3=A$, $B_2=B$, $B_1=C$, $B_0=0$。所以二进制数 $B = (ABC0)_2$。
    
- **$CI$（进位）：** 和 $B_0$ 连在一起，也是 $0$。
![[dcb0144f5d80234fe0f6182509ed7df6.png]]
**它的数学实质：**

在二进制中，将 $(ABC)_2$ 后面补个 $0$ 变成 $(ABC0)_2$，等价于**乘以 2**（左移一位）。

而 $(0ABC)_2$ 就是原数本身。

所以这个电路实现的运算是：

$$D = A + B = (ABC)_2 \times 2 + (ABC)_2 = 3 \times (ABC)_2$$

这是一个**三倍频 / 三倍量运算电路**。输出的 $D_4 D_3 D_2 D_1 D_0$ 就是输入 $C B A$ 乘以 3 后的二进制结果。






## BCD码（Binary-Coded Decimal)
用二进制编码的十进制数

它们的核心目的，是为了**用 4 位二进制数来表示 0~9 这十个十进制数字**，方便计算机处理人类习惯的十进制输入和输出。

### 1. 8421码（恒权码）

8421码是应用最广的 BCD 码，它的特点是“所见即所得”。

- **为什么叫 8421？** 因为这 4 个二进制位从高到低，每一位的权值（权重）分别是 $8$、$4$、$2$、$1$。
    
    对于一个 4 位二进制数 $D_3 D_2 D_1 D_0$，它代表的十进制数计算公式为：
    
    $$\text{十进制值} = D_3 \times 8 + D_2 \times 4 + D_1 \times 2 + D_0 \times 1$$
    
- **举个例子：**
    
    十进制的 `5`，用 8421 码表示就是 `0101`（因为 $0\times8 + 1\times4 + 0\times2 + 1\times1 = 5$）。这和普通的二进制转换完全一样。
    
- **致命缺点：**
    
    当两个 8421 码相加时，如果结果大于 9（例如 $5+6=11$，二进制为 `1011`），由于 8421 码只定义了 0~9（`0000` 到 `1001`），`1011` 在 8421 码里是**非法码**。这时候电路必须执行复杂的“加 6 修正”才能正确向高位进位。
    

### 2. 余3码（非恒权码）

余3码就是为了解决上述加法进位麻烦而设计的一种**变相编码**。

- **什么是余3码？**
    
    顾名思义，就是在 8421 码（或者说原十进制数）的基础上，**多出（余）3**。
    
    $$\text{余3码} = \text{十进制数} + 3 \text{ 的二进制表示}$$
    
- **举个例子：**
    
    想表示十进制的 `5`：
    
    先算 $5 + 3 = 8$，再把 8 转成二进制 `1000`。所以，十进制 `5` 的余3码就是 `1000`。
    
- **它的每一位有权重吗？**
    
    没有。你不能再用 $8, 4, 2, 1$ 去乘它的每一位，所以它叫**非恒权码**。


### 🎯 为什么要发明“余3码”？

既然 8421 码这么直观，为什么还要学令人绕弯子的余3码？因为它有两个在电路设计上非常绝妙的特性：

#### ① 完美的“互补性”（对九求补）

观察上面的表格：

- 十进制的 `0`（余3码 `0011`）和 `9`（余3码 `1100`），它们俩的二进制位**正好逐位取反**！
    
- 同理，`1`（`0100`）和 `8`（`1011`）也正好互反；`2` 和 `7`、`3` 和 `6`、`4` 和 `5` 都是如此。
    
- **好处：** 在数字电路中，如果要算十进制的减法（求补码），**只需要接一个反相器（非门）把余3码的所有位取反，就直接得到了该数对 9 的补数**。这让减法器的设计变得极其简单。
    

#### ② 自动生成进位信号

- 在十进制中，$5 + 5 = 10$，应该产生进位。
    
- 来看余3码相加：`5` 的余3码是 `1000`，`1000 + 1000 = 10000`。
    
- 看到最高位的 `1` 了吗？4 位全加器在算 $5+5$ 的余3码时，**它的最高进位输出引脚（$CO$）会自动跳变成 1**！
    
- **好处：** 它不需要像 8421 码那样用额外的逻辑门去判断“结果是不是大于 9”，全加器自身的进位输出就是十进制的进位信号，极大地简化了多位加法电路。





## 8421BCD码 转换成 余3码

- **概念复习：** 余3码就是把 8421BCD码 的数值**加上 3** 得到的代码。
    
- **公式：** $Y_3 Y_2 Y_1 Y_0 = D_3 D_2 D_1 D_0 + 0011$

![[1a714906f47221dd45c9ce374e0ffdd1.png|1018]]

**连线方法：**

1. 把输入的 8421 码 $D_3 D_2 D_1 D_0$ 直接送入 $A$ 端（$A_3 A_2 A_1 A_0$）。
    
2. 要在 $B$ 端固定输入二进制的 3，即 $(0011)_2$：
    
    - 高位 $B_3, B_2$ 接 **0（接地/低电平）**。
        
    - 低位 $B_1, B_0$ 连在一起接 **1（接电源/高电平）**。
        
3. 没有来自更低位的进位，所以 $CI$ 也要**接地（0）**。
    
4. 最后的输出 $Y_3 Y_2 Y_1 Y_0$ 就是直出的和 $S_3 S_2 S_1 S_0$。




## 用 74LS283 实现将余3码转换成 8421BCD码

![[02c8e0b0af40b81494226d5c0a7a5869.png|802]]

### 第一步：列出数学关系式

余3码比 8421 码大 3。现在已知输入是余3码（假设为 $X_3 X_2 X_1 X_0$），要求输出 8421 码（假设为 $Y_3 Y_2 Y_1 Y_0$）。

明显的，我们需要做**减法**：

$$Y = X - 3 = X - (0011)_2$$

### 第二步：把减法变成加法（数电核心技巧）

加法器只能做加法，怎么做减法？在计算机和数字电路中，我们利用**补码**将减法转为加法。

对 4 位二进制数而言，减去 $3$ 等价于加上 $-3$ 的补码。

- $-3$ 的 4 位补码是多少？
    
- 先取反再加1：$(0011) \rightarrow 翻转 \rightarrow (1100) \rightarrow 加1 \rightarrow (1101)_2$
    
- 所以，$X - 3 \equiv X + 13 \equiv X + (1101)_2 \pmod{16}$
    

### 第三步：确定引脚连线

依据 $Y = X + (1101)_2 + 0$：

1. **$A$ 端（输入变数）：**
    
    - 接输入的余3码信号。令 $A_3=X_3, A_2=X_2, A_1=X_1, A_0=X_0$。
        
2. **$B$ 端（固定常量 $(1101)_2$）：**
    
    - $B_3$ 接 **1** (高电平)
        
    - $B_2$ 接 **1** (高电平)
        
    - $B_1$ 接 **0** (低电平/地)
        
    - $B_0$ 接 **1** (高电平)
        
3. **$CI$ 端（进位输入）：**
    
    - 由于我们在 $B$ 端已经直接使用了完整的补码 `1101`，不需要额外的末位加1，所以 $CI$ 直接**接 0（地）**。
        
4. **输出端：**
    
    - $S_3 S_2 S_1 S_0$ 即为转换后的 8421BCD码 输出。
        
    - 最高位进位 $CO$ 悬空（不用管它，因为在 BCD 码有效范围内，这个减法转化出的加法溢出进位无实际意义）。
