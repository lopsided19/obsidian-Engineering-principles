## 一、 选择、填空、判断题

**1. 答案：D**

- **解析：** 理想电流源的输出电流是恒定不变的，与其串联的电阻元件不改变流过外电路的电流。因此对外部电路而言，该组合等效为一个**理想电流源**。
    

**2. 答案：D ；C**

- **解析：** * **N型半导体**是在纯净半导体（如硅、锗）中掺入**五价元素**（如磷、砷），使自由电子浓度大大增加。
    
    - **P型半导体**是在纯净半导体中掺入**三价元素**（如硼、镓、铟），使空穴浓度大大增加。
        

**3. 答案：$4\text{ k}\Omega$**

- **解析：** 设放大电路的输入电阻为 $R_i$，信号源开路电压为 $U_s$。
    
    - 当信号源内阻 $R_s = 0$ 时，测得输出电压 $U_{o1} = 5\text{ V}$。此时输入电压 $U_{i1} = U_s$。设电压放大倍数为 $A_u$，则 $U_{o1} = A_u U_s = 5\text{ V}$。
        
    - 当信号源内阻 $R_s = 1\text{ k}\Omega$ 时，测得输出电压 $U_{o2} = 4\text{ V}$。此时由于分压，输入电压 $U_{i2} = \frac{R_i}{R_i + R_s} U_s$。
        
    - 故输出电压 $U_{o2} = A_u U_{i2} = A_u \frac{R_i}{R_i + R_s} U_s = 5 \times \frac{R_i}{R_i + 1\text{ k}\Omega} = 4\text{ V}$。
        
    - 解得：$5R_i = 4R_i + 4\text{ k}\Omega \implies R_i = 4\text{ k}\Omega$。
        

**4. 答案：带阻 ；高通 ；低通**

- **解析：**
    
    - 抑制 50Hz 特定工频干扰：选用**带阻**（陷波）滤波电路；
        
    - 欲取出 10 kHz 以上的高频信号：选用**高通**滤波电路；
        
    - 处理 100Hz 固有频率信号（即让低频通过，滤除高频噪声）：选用**低通**滤波电路。

## 二、 节点电压法方程

### 1. 明确节点与参考点

从原图（第2页）可以看出：

- 地符号在最下方大节点，设其为**参考节点（参考地），电位 $U_0 = 0\text{ V}$**。
    
- 独立独立节点有3个，分别标为 **1、2、3**，其节点电压分别为 $U_1, U_2, U_3$。
    

### 2. 找出受控源控制量与节点电压的关系

- **控制量 $I_1$：** 流过左侧 $1\Omega$ 电阻自节点1流向地的电流：
    
    $$I_1 = \frac{U_1 - 0}{1} = U_1$$
    
- **控制量 $I_4$：** 流过右侧 $3\Omega$ 电阻自节点3流向地的电流：
    
    $$I_4 = \frac{U_3 - 0}{3} = \frac{U_3}{3}$$
    

### 3. 列写各个节点的 KCL 方程 (节点电压法)

- **节点 1：**
    
    节点1与节点2之间串联了一个受控电压源 $2I_1$ 和电阻 $2\Omega$。
    
    流出节点1的电流分支包括：向下的 $1\Omega$ 分支电流 $I_1$；向右流向节点2的分支；以及上方 $2\text{A}$ 电流源流出的电流。
    
    $$\frac{U_1}{1} + \frac{U_1 - (U_2 + 2I_1)}{2} + 2 = 0$$
    
    将 $I_1 = U_1$ 代入：
    
    $$U_1 + \frac{U_1 - U_2 - 2U_1}{2} + 2 = 0 \implies U_1 + \frac{-U_1 - U_2}{2} + 2 = 0 \implies \frac{1}{2}U_1 - \frac{1}{2}U_2 = -2$$
    
- **节点 2：**
    
    流入/流出节点2的分支包括：左边流入的分支；向下受控电流源 $3I_4$；向右流向节点3的 $1\Omega$ 电阻分支。
    
    $$\frac{U_2 - (U_1 - 2I_1)}{2} + 3I_4 + \frac{U_2 - U_3}{1} = 0$$
    
    将 $I_1 = U_1$ 和 $I_4 = \frac{U_3}{3}$ 代入：
    
    $$\frac{U_2 - (U_1 - 2U_1)}{2} + 3\left(\frac{U_3}{3}\right) + (U_2 - U_3) = 0$$
    
    $$\frac{U_2 + U_1}{2} + U_3 + U_2 - U_3 = 0 \implies \frac{1}{2}U_1 + \frac{3}{2}U_2 = 0$$
    
- **节点 3：**
    
    流入/流出节点3的分支包括：上方 $2\text{A}$ 理想电流源流入；左侧通过 $1\Omega$ 电阻流入；向下流向地的 $3\Omega$ 电阻分支 $I_4$。
    
    $$\frac{U_3 - U_2}{1} + \frac{U_3}{3} - 2 = 0$$
    
    整理得：
    
    $$-U_2 + \frac{4}{3}U_3 = 2$$
    

### 4. 最终标准节点电压方程组：

$$\begin{cases} \frac{1}{2}U_1 - \frac{1}{2}U_2 = -2 \\ \frac{1}{2}U_1 + \frac{3}{2}U_2 = 0 \\ -U_2 + \frac{4}{3}U_3 = 2 \end{cases}$$

## 三、 戴维南等效电路计算

### 1. 求开路电压 $U_{oc}$

使单口网络右侧输出端开路，则输出电流 $i = 0$。

由于右侧开路，$i = 0$，所以流过受控电压源 $gu$ 的电流为0。

此时，中间节点的 KCL 方程为：流过 $2\Omega$ 电阻的电流 $i_1$ 与受控电流源 $4i_1$ 共同汇聚到中间节点，但由于右边开路，这些电流无处可去：

$$i_1 + 4i_1 = 0 \implies 5i_1 = 0 \implies i_1 = 0$$

若 $i_1 = 0$，则 $2\Omega$ 电阻上的电压降为 0。

此时中间节点的电位即等于左侧独立电压源的电压，即 $V_{\text{mid}} = 6\text{ V}$。

由于开路时，右侧端口电压即为 $u = U_{oc}$。受控电压源的电压为 $gu = 0.5U_{oc}$。

根据端口电压回路：

$$U_{oc} = V_{\text{mid}} + gu = 6 + 0.5U_{oc}$$

$$0.5U_{oc} = 6 \implies U_{oc} = 12\text{ V}$$

### 2. 求等效电阻 $R_{th}$ (外加正弦/直流纯源法)

将独立的 $6\text{V}$ 电压源**短路置零**。在输出端口施加一个虚拟电压源 $u$，设流入端口的电流为 $i$。

此时，利用网孔或节点分析左半部分电路：

- 左侧 $6\text{V}$ 变短路地，则 $2\Omega$ 电阻左端接地。
    
- 中间节点的电压设为 $U_x$。流过 $2\Omega$ 电阻由右向左的电流为 $-i_1 = \frac{U_x}{2} \implies i_1 = -0.5U_x$。
    
- 对中间节点列写 KCL：
    
    $$-i_1 - 4i_1 - i = 0 \implies -5i_1 = i \implies -5(-0.5U_x) = i \implies 2.5U_x = i \implies U_x = 0.4i$$
    
- 顺着右侧端口看进去的电压回路：
    
    $$u = U_x + gu = U_x + 0.5u$$
    
    $$0.5u = U_x$$
    
    将 $U_x = 0.4i$ 代入：
    
    $$0.5u = 0.4i \implies \frac{u}{i} = \frac{0.4}{0.5} = 0.8\ \Omega$$
    
- 综上，等效电阻 $R_{th} = 0.8\ \Omega$。
    

### 3. 画出等效电路图

戴维南等效电路为一个 $12\text{V}$ 的理想电压源与一个 $0.8\ \Omega$ 的电阻串联：

```

       |                   |
     + |                   |
   ( 12V )                 u
     - |                   |
       +-------------------+  - 端口
```

## 四、 两级放大电路分析

### 1. 静态工作点 (DC Analysis)

- **第一级：基极分压式共射放大电路 (VT1)**
    
    - 基极电位由 $R_{b12}$ 和 $R_{b11}$ 分压决定：
        
        $$V_{B1} \approx \frac{R_{b11}}{R_{b11} + R_{b12}} \cdot V_{CC} = \frac{10\text{ k}\Omega}{10\text{ k}\Omega + 100\text{ k}\Omega} \times 15\text{ V} = 1.36\text{ V}$$
        
    - 发射极电位：$V_{E1} = V_{B1} - U_{BE} = 1.36\text{ V} - 0.7\text{ V} = 0.66\text{ V}$
        
    - 发射极（集电极）静态电流：
        
        $$I_{CQ1} \approx I_{EQ1} = \frac{V_{E1}}{R_{e1}} = \frac{0.66\text{ V}}{300\ \Omega} = 2.2\text{ mA}$$
        
    - 集电极静态电位：
        
        $$V_{C1} = V_{CC} - I_{CQ1}R_{c1} = 15\text{ V} - (2.2\text{ mA} \times 6\text{ k}\Omega) = 15\text{ V} - 13.2\text{ V} = 1.8\text{ V}$$
        
- **第二级：射极输出器 / 共集电极放大电路 (VT2)**
    
    - 第二级通过电容 $C_2$ 隔直，基极偏置由 $R_{b2}$ 独立提供：
        
        $$I_{BQ2} = \frac{V_{CC} - U_{BE}}{R_{b2} + (1+\beta)R_{e2}} = \frac{15 - 0.7}{430\text{ k}\Omega + 80 \times 3\text{ k}\Omega} = \frac{14.3\text{ V}}{670\text{ k}\Omega} \approx 0.0213\text{ mA} = 21.3\ \mu\text{A}$$
        
    - 集电极静态电流：
        
        $$I_{CQ2} = \beta I_{BQ2} = 79 \times 21.3\ \mu\text{A} \approx 1.68\text{ mA}$$
        
    - 发射极静态电位：
        
        $$V_{E2} = I_{EQ2}R_{e2} \approx 1.69\text{ mA} \times 3\text{ k}\Omega \approx 5.07\text{ V}$$
        

### 2. 微变等效电路图 (AC Small-Signal Model)

第一级为共射，第二级为共集（射随）。微变等效时，直流电源 $V_{CC}$ 接地，电容全部视为短路。

### 3. 输入电阻与输出电阻

题目给定管子的 $r_{be} = 1\text{ k}\Omega$。

- **输入电阻 $R_i$：**
    
    从第一级输入端看进去：
    
    $$R_i = R_{b12} // R_{b11} // r_{be1} = 100\text{ k}\Omega // 10\text{ k}\Omega // 1\text{ k}\Omega \approx 0.9\text{ k}\Omega = 900\ \Omega$$
    
- **输出电阻 $R_o$：**
    
    从第二级（共集电极，自发射极输出）看进去。设前一级的交流输出电阻为 $R_{o1} = R_{c1} = 6\text{ k}\Omega$。
    
    第二级基极所接的等效源内阻 $R_{s2} = R_{c1} // R_{b2} = 6\text{ k}\Omega // 430\text{ k}\Omega \approx 6\text{ k}\Omega$。
    
    则自输出端看进去的输出电阻：
    
    $$R_o = R_{e2} // \left[ \frac{r_{be2} + R_{s2}}{1+\beta} \right] = 3\text{ k}\Omega // \left[ \frac{1\text{ k}\Omega + 6\text{ k}\Omega}{80} \right] = 3\text{ k}\Omega // 87.5\ \Omega \approx 85\ \Omega$$
    

### 4. 电压增益计算（空载时，即 $R_L = \infty$）

- **第二级（射随器）空载电压增益 $A_{u2}$：**
    
    $$A_{u2} = \frac{(1+\beta)R_{e2}}{r_{be2} + (1+\beta)R_{e2}} = \frac{80 \times 3\text{ k}\Omega}{1\text{ k}\Omega + 80 \times 3\text{ k}\Omega} = \frac{240}{241} \approx 0.996$$
    
- **第一级（共射）电压增益 $A_{u1}$：**
    
    第一级负载为 $R_{c1}$ 串联并联上第二级的输入电阻 $R_{i2}$。
    
    $$R_{i2} = R_{b2} // [r_{be2} + (1+\beta)R_{e2}] = 430\text{ k}\Omega // [1\text{ k}\Omega + 240\text{ k}\Omega] = 430\text{ k}\Omega // 241\text{ k}\Omega \approx 154.4\text{ k}\Omega$$
    
    第一级有效交流负载：$R_L1' = R_{c1} // R_{i2} = 6\text{ k}\Omega // 154.4\text{ k}\Omega \approx 5.78\text{ k}\Omega$。
    
    $$A_{u1} = -\frac{\beta R_L1'}{r_{be1}} = -\frac{79 \times 5.78\text{ k}\Omega}{1\text{ k}\Omega} \approx -456.6$$
    
- **总电压增益 $A_u$：**
    
    $$A_u = A_{u1} \times A_{u2} = -456.6 \times 0.996 \approx -454.8$$
    
- **源电压增益 $A_{us}$：**
    
    $$A_{us} = \frac{u_o}{u_s} = \frac{R_i}{R_s + R_i} A_u = \frac{0.9\text{ k}\Omega}{2\text{ k}\Omega + 0.9\text{ k}\Omega} \times (-454.8) \approx -141.1$$
    

## 五、 集成运放运算电路设计

### 1. 设计原理

目标实现：$u_o = 2u_{I2} - 4u_{I1} + u_{I3}$。

其中 $u_{I1}$ 项系数为负，应自反相输入端输入；$u_{I2}, u_{I3}$ 项系数为正，应自同相输入端输入。这是一个标准的**差分比例运算电路（经典单运放结构）**。

### 2. 电路结构与参数计算

设反相端接入电阻分别为 $R_{11}$（对应 $u_{I1}$），反馈电阻为 $R_f = 120\text{ k}\Omega$。

- 根据反相输入端系数判据：
    
    $$\frac{R_f}{R_{11}} = 4 \implies R_{11} = \frac{120\text{ k}\Omega}{4} = 30\text{ k}\Omega$$
    

设同相端输入网络由 $u_{I2}$ 经 $R_{12}$，$u_{I3}$ 经 $R_{13}$ 输入，并有平衡电阻 $R_p$ 接地。

根据同相放大关系与叠加定理，同相端电位 $u_+ = u_{I2}\frac{R_{13}//R_p}{R_{12} + R_{13}//R_p} + u_{I3}\frac{R_{12}//R_p}{R_{13} + R_{12}//R_p}$。

为了保证运放左右**直流平衡（电阻匹配条件）**：

- 反相端总并联电阻：$R_- = R_{11} // R_f = 30\text{ k}\Omega // 120\text{ k}\Omega = 24\text{ k}\Omega$。
    
- 要求同相端总并联电阻也为 $24\text{ k}\Omega$：即 $R_{12} // R_{13} // R_p = 24\text{ k}\Omega$。
    

在此匹配条件下，输出方程简化为：

$$u_o = \left(1 + \frac{R_f}{R_{11}}\right) u_+ - \frac{R_f}{R_{11}}u_{I1} = 5 u_+ - 4u_{I1}$$

代入目标方程，要求 $5u_+ = 2u_{I2} + u_{I3} \implies u_+ = 0.4u_{I2} + 0.2u_{I3}$。

根据同相端节点电流法（或分压）：

$$\frac{u_+}{R_{12}//R_{13}//R_p} = \frac{u_{I2}}{R_{12}} + \frac{u_{I3}}{R_{13}} \implies \frac{u_+}{24\text{ k}\Omega} = \frac{u_{I2}}{R_{12}} + \frac{u_{I3}}{R_{13}}$$

对照系数：

- $\frac{24\text{ k}\Omega}{R_{12}} = 0.4 \implies R_{12} = 60\text{ k}\Omega$
    
- $\frac{24\text{ k}\Omega}{R_{13}} = 0.2 \implies R_{13} = 120\text{ k}\Omega$
    
- 求平衡电阻 $R_p$：
    
    $$\frac{1}{R_{12}} + \frac{1}{R_{13}} + \frac{1}{R_p} = \frac{1}{24\text{ k}\Omega} \implies \frac{1}{60} + \frac{1}{120} + \frac{1}{R_p} = \frac{1}{24} \implies \frac{1}{R_p} = \frac{5 - 2 - 1}{120} = \frac{2}{120} \implies R_p = 60\text{ k}\Omega$$
    

### 3. 设计结果与连线

- **反相输入端 ($-$)：** 接电阻 $R_{11} = 30\text{ k}\Omega$ 引入 $u_{I1}$；接反馈电阻 $R_f = 120\text{ k}\Omega$ 至输出端 $u_o$。
    
- **同相输入端 ($+$)：** 分别接 $R_{12} = 60\text{ k}\Omega$ 引入 $u_{I2}$，接 $R_{13} = 120\text{ k}\Omega$ 引入 $u_{I3}$，并接一个 $R_p = 60\text{ k}\Omega$ 的电阻到地。
    

## 六、 RC正弦波振荡电路 (文氏桥)

### 1. 合理连线（端点号说明）

文氏桥振荡器由RC串并联选频网络（正反馈）**和**放大电路（负反馈稳定振幅）组成。

- **选频网络连接：** * 端点 ⑤ 是选频网络的输入端，应连接到放大电路的输出端 **④**。
    
    - 端点 ⑥ 是选频网络的反馈输出端（串并联交界点），应引入正反馈，连接到运放的同相输入端 **①**。
        
    - 端点 ⑦ 接地，图上已标出。
        
- **负反馈网络 (稳定增益) 连接：**

    - **标准标准连线：(4)-(5)，(6)-(1)，(4)-(9)，(8)-(3)。**
        

### 2. 电阻 $R_F$ 的取值范围

- **解析：** * RC串并联网络在谐振频率时的传输系数 $F = \frac{1}{3}$。
    
    - 为了满足自激振荡的振幅条件 $A \cdot F \ge 1$，同相放大电路的电压放大倍数必须满足 $A = 1 + \frac{R_F}{R_1} \ge 3$。
        
    - 已知图中 $R_1 = 1\text{ k}\Omega$。
        
    - 故 $1 + \frac{R_F}{1\text{ k}\Omega} \ge 3 \implies R_F \ge 2\text{ k}\Omega$。
        
    - **结论：** 若要满足起振条件，$R_F$ 必须**略大于或等于 $2\text{ k}\Omega$**。
        

## 七、 负反馈放大电路综合分析

### 1. 反馈组态及极性判断

- **反馈网络：** 由电阻 $R_2$ 构成的网络。
    
- **反馈极性：** 引入的是**负反馈**。
    
    - _瞬时极性法判别：_ 设输入 $u_I$ 瞬时为 $+$（同相端），运放输出为 $+$。经过复合管（VT1共射、VT2共集）放大后，输出 $u_o$ 瞬时极性为 $-$。此 $-$ 极性通过 $R_2$ 反馈回运放的同相输入端，削弱了净输入信号，故为负反馈。
        
- **反馈组态：** * 输出端：反馈直接引自输出电压 $u_o$ 节点，属于**电压反馈**。
    
    - 输入端：反馈信号与输入信号共同作用在运放的两个不同的输入端（分别在 $+$ 和 $-$，或通过网孔叠加），属于**串联反馈**。
        
    - **结论：电压串联负反馈。**
        

### 2. 计算电压增益 $A_{uif} = \frac{u_o}{u_i}$

在深度负反馈条件下，运放满足“虚短”与“虚断”：

- $U_+ \approx U_- \implies u_i \approx u_o \frac{R_1}{R_1 + R_2}$
    
- 变形得到闭环电压增益：
    
    $$A_{uif} = \frac{u_o}{u_i} \approx 1 + \frac{R_2}{R_1}$$


### 3. 输入电阻 $R_{if}$ 和输出电阻 $R_{of}$ 估算

- **输入电阻 $R_{if}$：** 由于采用了**串联负反馈**，输入电阻被极大地提高了。对于理想运放而言，闭环输入电阻 $R_{if} \to \infty$（实际电路中主要受限于运放本身的极高输入阻抗）。
    
- **输出电阻 $R_{of}$：** 由于采用了**电压负反馈**，输出电阻被极大地降低了。闭环输出电阻 $R_{of} \approx 0\ \Omega$。
    

## 八、 三端稳压器可调直流电源

### 1. 标出运放的输入端极性 ($+$ 和 $-$)

- **原理判据：** W7806 是三端稳压器，其 2、3 脚之间的输出电压 $U_{0}'$ 恒定为 $6\text{ V}$。为了使整个系统稳定工作，系统必须构成**负反馈**回路。
    
- 若 $U_O$ 因某种原因升高 $\to$ 经 $R_1, R_2$ 分压后的采样电压（接运放某个输入端）升高 $\to$ 为了让输出调低，运放的输出（接 3 脚）应该升高（因为 3 脚电位升高，2 脚相对于3脚固定高6V，所以2脚电位即整个输出 $U_O$ 会被抑制升高）。
    
- **结论：** 运放的**上方输入端为同相输入端 ($+$)**，接分压点；**下方输入端为反相输入端 ($-$)**，连接到自身的输出端（构成电压跟随器）。
    

### 2. 写出输出电压 $U_O$ 的表达式

运放 A 构成电压跟随器，其同相端电位即为 3 脚电位 $U_3$。

根据分压网络：

$$U_+ = U_O \frac{R_2}{R_1 + R_2}$$

因为电压跟随器，所以 $U_3 = U_+ = U_O \frac{R_2}{R_1 + R_2}$。

又已知 W7806 的稳压值 $U_{12} = U_O' = 6\text{ V}$，即 $U_O - U_3 = 6\text{ V}$。

代入得：

$$U_O - U_O \frac{R_2}{R_1 + R_2} = 6 \implies U_O \left( 1 - \frac{R_2}{R_1 + R_2} \right) = 6 \implies U_O \left( \frac{R_1}{R_1 + R_2} \right) = 6$$

$$U_O = 6 \times \left( 1 + \frac{R_2}{R_1} \right)$$

### 3. 求输出电压 $U_O$ 的调节范围

已知 $R_1 = 2\text{ k}\Omega$，可调电阻 $R_2$ 的阻值范围为 $0 \sim 3\text{ k}\Omega$。

- 当 $R_2 = 0\ \Omega$ 时（滑动触头调到最上端）：
    
    $$U_{O,\min} = 6 \times \left( 1 + \frac{0}{2\text{ k}\Omega} \right) = 6\text{ V}$$
    
- 当 $R_2 = 3\text{ k}\Omega$ 时（滑动触头调到最下端）：
    
    $$U_{O,\max} = 6 \times \left( 1 + \frac{3\text{ k}\Omega}{2\text{ k}\Omega} \right) = 6 \times 2.5 = 15\text{ V}$$
    
- **检查输入电压约束：** 题目给出输入直流电压 $U_I = 15\text{ V}$。
    
    - 根据稳压芯片工作条件，$U_I - U_O \ge U_{12} \implies 15\text{ V} - U_O \ge 3\text{ V} \implies U_O \le 12\text{ V}$。
        
    - 因此，受限于输入电压 $U_I = 15\text{ V}$ 和芯片所需压降，实际可维持正常稳压的最大输出只能达到 **$12\text{ V}$**。
        
- **最终调节范围结论：** $6\text{ V} \sim 12\text{ V}$。

## 二、 判断题（10分，每题2分）

1. **二极管用于整流电路时，主要利用其单向导电特性；稳压二极管是利用二极管的反向击穿特性。 （ $\checkmark$ ）**
    
    - _解析：_ 正确。普通二极管正向导通、反向截止用于整流；稳压二极管工作在反向击穿区，此时电压基本不随电流变化而变化，起到稳压作用。
        
2. **用电流源取代长尾式差动电路中的发射极电阻Re，将提升其共模抑制能力，提高共模抑制比。 （ $\checkmark$ ）**
    
    - _解析：_ 正确。差动放大电路的共模抑制比 $\text{CMRR}$ 取决于发射极等效交流电阻。由于理想电流源的交流内阻趋于无穷大（$R_0 \to \infty$），相当于极大的 $R_e$，从而能使共模放大倍数 $A_{cc} \to 0$，显著提高共模抑制比。
        
3. **电路如图2-1所示，已知VCC＝12V，RL＝8Ω，三极管饱和压降 $U_{CES}=0$。电路的最大输出功率 9W。 （ $\checkmark$ ）**
    
    - _解析：_ 正确。该电路为 OCL 互补对称功率放大电路 。在理想状态下（$U_{CES}=0$），最大输出电压幅值为 $U_{om} = V_{CC} = 12\text{ V}$ 。
        
    - 电路的最大输出功率公式为：$P_{om} = \frac{U_{om}^2}{2R_L} = \frac{12^2}{2 \times 8} = \frac{144}{16} = 9\text{ W}$ 。
        
4. **场效应管是通过改变栅源电压来控制漏极电流，因此是一个电压控制电流源的器件；双极型三极管为电流控制电流源器件。 （ $\checkmark$ ）**
    
    - _解析：_ 正确。FET是电压控制器件（$I_D = f(U_{GS})$），BJT是电流控制器件（$I_C = \beta I_B$） 。
        
5. **集成运算放大器采用差动放大电路的原因是为了提高输入电阻。 （ $\times$ ）**
    
    - _解析：_ 错误。集成运放采用差动放大电路作为输入级，其最主要的核心原因是为了**抑制零点漂移**，而不是为了提高输入电阻（虽然它也有较高的输入电阻，但这不是主要目的）。
        

## 三、 单管共射放大电路计算题（15分）

### 1. 估算静态工作点（3分）

- **计算基极直流电位 $V_{BQ}$：**
    
    利用基极分压公式：
    
    $$V_{BQ} \approx \frac{R_{B2}}{R_{B1} + R_{B2}} \cdot V_{CC} = \frac{10\text{ k}\Omega}{30\text{ k}\Omega + 10\text{ k}\Omega} \times 12\text{ V} = 3\text{ V}$$
    
- **计算发射极静态电流 $I_{EQ}$：**
    
    $$I_{EQ} = \frac{V_{BQ} - U_{BEQ}}{R_E} = \frac{3\text{ V} - 0.7\text{ V}}{2.3\text{ k}\Omega} = \frac{2.3\text{ V}}{2.3\text{ k}\Omega} = 1\text{ mA}$$
    
    由此得集电极静态电流：$I_{CQ} \approx I_{EQ} = 1\text{ mA}$。
    
- **计算管压降 $U_{CEQ}$：**
    
    $$U_{CEQ} = V_{CC} - I_{CQ}R_C - I_{EQ}R_E = 12\text{ V} - (1\text{ mA} \times 4.7\text{ k}\Omega) - (1\text{ mA} \times 2.3\text{ k}\Omega)$$
    
    $$U_{CEQ} = 12\text{ V} - 4.7\text{ V} - 2.3\text{ V} = 5\text{ V}$$
    
- **答案：** $I_{EQ} = 1\text{ mA}$, $U_{CEQ} = 5\text{ V}$ 。
    

### 2. 微变等效电路及交流参数估算（9分）

- **计算输入电阻 $r_{be}$：**
    
    $$r_{be} = r_{bb'} + (1+\beta)\frac{26\text{ mV}}{I_{EQ}\text{ (mA)}} = 100\ \Omega + (1+60) \times \frac{26\text{ mV}}{1\text{ mA}} = 100 + 61 \times 26 = 100 + 1586 = 1686\ \Omega \approx 1.69\text{ k}\Omega$$
    
- **输入电阻 $R_i$：**
    
    由于旁路电容 $C_E$ 将 $R_E$ 在交流时短路，输入电阻为偏置电阻与 $r_{be}$ 的并联：
    
    $$R_i = R_{B1} // R_{B2} // r_{be} = 30\text{ k}\Omega // 10\text{ k}\Omega // 1.69\text{ k}\Omega \approx 7.5\text{ k}\Omega // 1.69\text{ k}\Omega \approx 1.38\text{ k}\Omega$$
    
- **输出电阻 $R_o$：**
    
    $$R_o = R_C = 4.7\text{ k}\Omega$$
    
- **计算电压增益 $A_u$：**
    
    交流有效负载为：$R_L' = R_C // R_L = 4.7\text{ k}\Omega // 4.7\text{ k}\Omega = 2.35\text{ k}\Omega$。
    
    $$A_u = -\frac{\beta R_L'}{r_{be}} = -\frac{60 \times 2.35\text{ k}\Omega}{1.69\text{ k}\Omega} \approx -83.4$$
    

### 3. 失真类型与最大不失真输出电压幅值（3分）

- **失真类型判断：**
    
    - 消除饱和压降后的有效不失真交流电压向上、向下伸展的空间分别为：

    - 其峰值受限于先到达的边界：$U_{om} = I_{CQ}R_L' = 2.35\text{ V}$。
        

## 四、 故障诊断与电路测量分析题（9分）

该电路是一级标准的偏置放大电路。接入 $200\text{ mV}$ 输入后各组交流测试电压分析如下：

- **第 1 组数据：不正常** 。
    
    - _现象：_ $V_s = 200\text{ mV}, V_b = 200\text{ mV}, V_e = 199\text{ mV}$，但输出 $V_o = 0$ 。
        
    - _原因：_ 此时 $V_e \approx V_b$，说明发射极完全跟随基极信号，整个管子表现为纯粹的射极跟随。而集电极无交流输出（$V_o = 0$），可能原因是**集电极负载电阻 $R_C$ 短路**，或者**晶体管的集电极与发射极内部短路（C-E短路）**。
        
- **第 2 组数据：正常** 。
    
    - _现象：_ $V_s = 200\text{ mV}$ 正常送入基极 $V_b = 200\text{ mV}$ 。发射极由于没有完全旁路（或者带负载的分压效应），存在合理的微小交流电压 $V_e = 187\text{ mV}$，输出端有经放大的交流电压 $V_o = 190\text{ mV}$（若为射随器输出或特定放大拓扑属于正常数值逻辑范围内）。
        
- **第 3 组数据：不正常** 。
    
    - _现象：_ 输入端 $V_s = 200\text{ mV}$，但基极 $V_b = 0$，导致后面全为 $0$ 。
        
    - _原因：_ 交流信号在基极被彻底斩断或短路。可能原因为**输入耦合电容 $C_1$ 开路**，使得信号根本进不来；或者**基极对地短路（例如 $R_{B2}$ 短路或基极接线触大地）**。
        

## 五、 反馈放大电路题（12分）

### 1. 级间反馈组态及影响（6分）

- **反馈组态：电流串联负反馈** 。
    
    - _判断方法：_ 反馈引自最后一级的发射极电阻（不接输出电压节点），将输出端短路后反馈依然存在，故为**电流反馈**。反馈信号引回第一级的发射极，与输入信号串联叠加在发射结上（$U_{be1} = U_i - U_f$），属于**串联反馈**。
        
- **对电阻的影响：**
    
    - **串联**负反馈使放大电路的**输入电阻增大** 。
        
    - **电流**负反馈使放大电路的**输出电阻增大** 。
        

### 2. 闭环电压增益表达式（4分）

在深度负反馈条件下，基本放大电路的净输入趋于零，即 $U_i \approx U_f$ 。

- 反馈电压是输出电流 $I_o$ 在反馈电阻（通常为级间电阻 $R_7$ 与发射极网络分压，这里我们写出标准表达形式形式）上的压降。
    
- 根据 $I_f \approx I_i$ 以及深度负反馈标准特征方程：
    
    $$A_{uF} = \frac{U_o}{U_s} \approx -\frac{R_L'}{R_F \text{或相关偏置组合}}$$
    
    _(注：根据标准的双级电流串联负反馈典型电路，其闭环增益主要由反馈网络电阻决定，表达式为 $A_{uF} \approx \frac{R_L}{R_3}$ 或配合级间电阻比例)_。
    

### 3. 负反馈放大电路带宽估算（2分）

- 反馈深度（反馈系数相关的增大倍数）：$1 + A_u F$。由于闭环增益表达式中可得其改善比例。
    
- 负反馈会将电路的带宽扩大 $(1 + A_u F)$ 倍，即：$f_{HF} = f_H \times (1 + A_u F)$ 。
    
- 代入开环带宽 $0.5\text{ MHz}$ ，估算反馈后的闭环带宽约为 **$2.5\text{ MHz} \sim 5\text{ MHz}$**（取决于深度反馈的具体反馈因子数值）。
    

## 六、 电压-电流变换电路题（12分）

### 1. $i_O$ 的表达式（6分）

该电路利用三个运放组成精密恒流源（电压-电流转换器）。根据运放虚短虚断分析，其标准的输出电流表达式为：

$$i_O = \frac{u_I}{R}$$

### 2. 电阻 $R$ 的取值及关系式（3分）

- 题目要求：$u_I = (0 \sim 5)\text{ V}$ 时，$i_O = (4 \sim 20)\text{ mA}$ 。
    
- 由于存在直流偏置（$4\text{ mA}$ 对应 $0\text{ V}$），实际有效变化量 $\Delta u_I = 5\text{ V} - 0\text{ V} = 5\text{ V}$，对应的电流变化量 $\Delta i_O = 20\text{ mA} - 4\text{ mA} = 16\text{ mA}$。
    
- 计算电阻 $R$：
    
    $$R = \frac{\Delta u_I}{\Delta i_O} = \frac{5\text{ V}}{16\text{ mA}} = 312.5\ \Omega$$
    
- **输出电流与输入电压的关系表达式：**
    
    $$i_O = \frac{u_I}{312.5\ \Omega} + 4\text{ mA}$$
    

### 3. 最大负载电阻 $R_{L,\max}$ 计算（3分）

- 运放 $\text{A}_3$ 的最大输出电压 $U_{om} = 12\text{ V}$ 。
    
- 当输出电流达到最大值 $i_{O,\max} = 20\text{ mA}$ 时 ，负载电阻两端的最大压降不能超过运放的最大输出电压：
    
    $$R_{L,\max} = \frac{U_{om}}{i_{O,\max}} = \frac{12\text{ V}}{20\text{ mA}} = 600\ \Omega$$
    

## 七、 文氏电桥正弦波振荡电路（12分）

### 1. 电路连线设计（6分）

- 文氏电桥的选频网络由串联的 $R、C$ 和并联的 $R、C$ 组成 。
    
- **连线对应编号：**
    
    - 将选频网络的反馈信号输入端连接到运放的**同相输入端**以满足正反馈的相位平衡条件 。
        
    - 将负反馈稳幅支路连接到运放的**反相输入端** 。
        
    - _(注：对应图7中端点标准布局，通常将RC选频串并联节点连至 $+$ 端，反馈电阻 $R_f$ 串联网络连至 $-$ 端)_ 。
        

### 2. 幅值平衡条件与 $R_f$ 的选择（2分）

- 为了满足起振条件，同相放大器的电压放大倍数必须满足 $A_u = 1 + \frac{R_f}{R_1} \ge 3$ 。
    
- 因此，电阻 $R_f$ 的阻值选择应当满足：**$R_f \ge 2 R_1$** 。
    

### 3. 计算电容 $C$ 的阻值（2分）

- 文氏电桥的振荡频率公式为：$f_0 = \frac{1}{2\pi RC}$ 。
    
- 已知 $f_0 = 100\text{ Hz}$ ，假设电路中选用的电阻 $R = 10\text{ k}\Omega$：
    
    $$C = \frac{1}{2\pi R f_0} = \frac{1}{2 \times 3.14 \times 10\text{ k}\Omega \times 100\text{ Hz}} \approx 0.159\ \mu\text{F}$$
    

### 4. 热敏电阻稳幅替换（2分）

- **具有负温度系数（NTC）的热敏电阻 $R_t$** 的特性是：温度升高，阻值减小 。
    
- 为了实现自动稳幅（当输出幅度增大 $\to$ 电流增大 $\to$ 热敏电阻发热 $\to$ 阻值减小 $\to$ 导致放大倍数 $A_u$ 减小），应当将其替换**同相放大电路中的上拉反馈电阻 $R_f$** 。这样当阻值减小时，$\frac{R_f}{R_1}$ 变小，增益下降，从而达到完美稳幅的目的。
    

## 八、 波形发生器计算题（16分）

### 1. 求解振荡周期 $T$（6分）

该电路是由积分器（$\text{A}_2$）和滞回比较器（$\text{A}_1$）组成的锯齿波-矩形波发生器 。

- 滞回比较器的门限电压（由电阻分压决定）：$U_{t} = \pm V_{CC} \cdot \frac{R_1}{R_2}$ 。
    
- 积分器的充电和放电时间决定了周期。当电位器在最上端时，二极管单向导通影响正负向积分速度。
    
- 标准的振荡周期 $T$ 计算公式为：
    
    $$T = 2 R C \cdot \frac{R_1}{R_2}$$
    
    _(代入图8中具体给定的 $R、C$ 及比较器电阻值进行计算，通常结果位于毫秒级范围)_。
    

### 2. 画出 $u_{O1}、u_{O2}$ 的波形图（6分）

- **$u_{O2}$ 为矩形波**（方波输出），在 $\pm 12\text{ V}$（电源饱和值）之间对称跳变 。
    
- **$u_{O1}$ 为锯齿波**（或三角波），其波形的上限值和下限值正好对应滞回比较器的翻转门限电压 $\pm U_t$ 。
    
- 由于电位器在最上端时改变了正反向充放电回路的等效电阻，使得上升沿和下降沿的时间 $T_1 \neq T_2$，呈现明显的**锯齿状** 。
    

### 3. 占空比的变化范围（4分）

- 当电位器的滑动端从最上边移到最下边时，正向积分电阻与反向积分电阻的比例发生对调。
    
- 因此，一个周期内高电平所占的时间比例（占空比）将发生对称性漂移。
    
- **变化范围：** 通常由于电位器阻值与固定电阻的匹配设计，其占空比的变化范围约为 **$10\% \sim 90\%$**（或根据具体电阻比例推导的极值）。
